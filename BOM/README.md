# BOM 
BOM(Browser Object Model) 是由浏览器提供的API，

提供了除了DOM以外的功能，比如

- 表示当前浏览器窗口的`window`
- 表示浏览记录的`history`
- 表示URL的`location`
- 表示用户代理的`navigator`
- 表示屏幕的`screen`
