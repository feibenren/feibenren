下面就是一个类似数组的对象。
```
var a = {
    0:'a',
    1:'b',
    2:'c',
    length:3
};

a[0] // 'a'
a[2] // 'c'
a.length // 3
```
上面代码的变量a是一个对象，但是看上去跟数组很像。所以只要有数字键和length属性，就是一个类似数组的对象。当然，变量a无法使用数组特有的一些方法，比如pop和push方法。而且，length属性不是动态值，不会随着成员的变化而变化。
