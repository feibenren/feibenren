# 各种坑

## xml格式
返回xml格式的时候，只有格式完全正确才可以正确返回，可用格式化工具检测

## jssdk signature
生成jssdk的时候，需要注意的是
timestamp：是秒数，不是毫秒数，默认生成的是毫秒数


## jssdk url
url一定要完整，正确
可以使用
```
location.href.split('#')[0]
```
## JS接口安全域名
不能含有http://,只能是一个域名，不能带有其他任何东西
比如:
- http://www.xxx.com
- http://www.xxx.com/
- xxx.xxx.com/
- xxx.xxx.com/index.php
以上都不正确
没有任何文档提示这点
