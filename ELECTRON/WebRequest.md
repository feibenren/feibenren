# WebRequest

在一个请求生命周期的不同阶段，截取和修改其内容。

使用 Session 的 WebRequest 属性访问 WebRequest 类的实例。

WebRequest 的方法接受可选的 filter 和 listener。 当 API 的事件发生时, 将使用 listener(details) 来调用 listener。 详细details 对象描述了该请求。
