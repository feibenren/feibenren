# ClipboardEvent

ClipboardEvent 接口描述了与修改剪切板相关的事件，这些事件包括 cut 、copy 和 paste 事件。

> 支持度不高

```
Event <--- ClipboardEvent 
```

# 属性

- ClipboardEvent.clipboardData
