# resize

resize可以设置一个元素`是否可以调整大小`

textarea默认可以调整大小，而resize属性可以自行设定


属性:

- vertival(只允许垂直方向拉伸)
- horiliztion
- both(两个方向都可以)
- none(初始值)

# overflow
resize命令依赖于overflow

想要resize生效，需要设置overflow不为visiable
