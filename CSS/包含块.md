# 包含块

绝大多数情况下，包含块就是自己的父元素

但是有一些情况下，不是


# 根元素

根元素的包含块叫做`出事包含块`，大小一般和`viewport`相同

## fixed 定位

它的contain box是当前的窗口


## 绝对定位

它的contain box是祖先元素中，距离自己最近的`position:relative/abslote/fixed`的元素


# 作用

计算定位和尺寸的时候，会根据包含块来计算
