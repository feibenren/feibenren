# 相对定位

不会脱离文档流

也就不会有重叠

# 计算后的值

相对定位`不会造成框被拆分或者拉伸`,但是绝对定位会

遵循这个规则
```
left = -right
```
top和bottom也一样

left的值和right的值，这种的时候有区别

####  left:auto，right:100px
按照规则算，left会是`-100px`,但是这个数值不会产生任何影响

####  left:100px，right:100px
两个都设置的话,那就必须舍弃一个，会按照`direction`属性来确定舍弃哪个
