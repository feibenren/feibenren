# clip
只有针对绝对定位的元素才有用

```
clip: rect(top  right  bottom  left);
```
这四个属性，和ps中的参考线作用类似，并且数值也是和参考线类似




[clip示例代码](https://jsbin.com/xunohuk/edit?html,css,output)



# 用途
比较少的使用，一般用在图片的裁剪效果上使用比较多
