# 布局

所谓的布局，是一种基于盒子与其兄弟和祖辈盒子的交互方式来确定盒子的位置和大小的算法

也就是说

`布局是一种算法`

一种`确定大小和位置`的算法

css中的`display`就是设置这种算法的命令

# dispaly

display CSS 属性指定了元素的显示类型

它包含两类基础特征

用于指定元素怎样生成盒模型

- 外部显示类型定义了元素怎样参与流式布局的处理
- 内部显示类型定义了元素内子元素的布局方式。

# 总结

display这个命令，用来`指定如何生成盒模型`
