# Headers

Headers接口是Fetch Api中的一部分，用来设置http headers



挂载在:
- Response.headers
- request.headers


方法:
- Headers.append():
- Headers.delete():
- Headers.get():
- Headers.has():
- Headers.values():
- Headers.set():

等等


# 总结
headers不是随便设置的，由于安全限制，有些是不允许设置的，或者说设置无效
