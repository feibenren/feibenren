## 闭包（closure）是定义在某个作用域中的函数


```
function a(){
    var b=function(){
        console.log('bbbbb');
    }
    return b;
}

```
# 闭包优势
- 安全，只有通过返回的对象才可以操作闭包中的内容
- 让变量始终保持在内存中




# 缺点
- 可能导致内存泄漏

# 为什么js会出现这种怪异的东西

作用域链，垃圾回收机制




# 闭包作用
- 减少全局变量
- 隐藏变量
=====>构造模块
