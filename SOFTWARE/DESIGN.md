# 软件设计

软件设计，是软件工程中的一个领域

可粗分为:

- 结构设计
- 数据设计
- 接口设计
- 过程设计

等不同子领域

简单说，就是绘制软件的`蓝图`

为了描述这个蓝图

提出了一个`视图`的概念

比如使用视图，来描述逻辑，描述结构等

---------------

还提出了一种`描述语言`，比如常见的`UML`


# 模块通信

软件被设计成为多个模块，这些模块之间如何通信，有多种模型

比如常见的

- C/S模型
- 插件模型
- 分层模型
- 事件模型

等等
