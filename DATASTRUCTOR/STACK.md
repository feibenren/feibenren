# STACK

栈也是一种线性表，只不过有特殊规定的线性表

线性表的规定是

- 有限
- 有序

栈的规定是
- 有限
- 有序
- 只能从一头进出
- 后入先出(LIFO):后进去的先出来

栈这种数据结构在计算机中用途很广泛，不仅仅在代码本身存在，在硬件方面也都大量使用

这里只说这种数据结构

```
------------------------
             [3] [2] [1]|
------------------------
```


比如`刚烙好的煎饼堆`,刚烙好的在最底下，要叠的时候，只能从最上面开始

#### 名词解释

- 栈底
- 栈顶
- 入栈/压栈
- 出栈/弹栈


# 进出

在栈这种数据结构中，没有增删改查等操作

它只有出栈和入栈


# 实现

实现方式可以使用`顺序表`，也可以使用`链表`实现
