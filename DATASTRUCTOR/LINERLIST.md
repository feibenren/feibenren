# 定义

一个有顺序的有限序列

有两点

- 有序的
- 有限

能达到这两点的，就算是一个线性表，其他的不在要求

比如

```
[]----[]---[]

```

这三个就是有序的，有限的数据结构，这就是一个线性表

比如佛珠，也可以算作一个线性表，满足这两个条件

---------------

细分:

- 顺序表
- 链表
    - 堆
    - 栈
- 堆
- 栈


--------------

还有几个专有名词

- 前驱:一个元素的前面的所有元素,可能多个
    - 直接前驱:排在一个元素前面的元素,最多就一个
- 后继
    - 直接后继


在链表中，只有开头和结尾没有可能没有前驱和后继，其他都有
