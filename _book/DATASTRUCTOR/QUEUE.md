# QUEUE

队列也是一种线性表，只不过有特殊规定的线性表



线性表的规定是

- 有限
- 有序

栈的规定是
- 有限
- 有序
- 只能从两头进出
- 先入先出(FIFO):先进去的先出来

队列这种数据结构使用也非常广泛

这里只说这种数据结构

```
------------------------
     [3] [2] [1]
------------------------
```

这种数据结构是在现实中对标与`排队买票的队列`，先排队的先买票


#### 名词解释

- 出队
- 入队
- 队头
- 队尾


# 进出

在队列这种数据结构中，没有增删改查等操作

它只有出队和入队

# 实现

实现方式可以使用`顺序表`，也可以使用`链表`实现
