# console

console是浏览器提供的，控制台的接口

> console这个对象不是ecma规定的，是一个web api

### 打印信息
- console.log()
- console.info()
- console.error()
- console.warn()
- console.debug()
- console.table()
- console.dir()

### 计数
- console.count()

### 计时
- console.time()
- console.timeEnd()

### 断言
- console.assert()

### 调试
- console.trace()
